# BK Connect App

Next.js frontend for BK Connect.

## Run

```bash
cp .env.example .env.local
npm install
npm run dev
```

The frontend runs on `http://localhost:3000`.

Set `NEXT_PUBLIC_API_URL` to the Nest API base URL.
Default local value:

```bash
NEXT_PUBLIC_API_URL=http://localhost:4000
```

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```
